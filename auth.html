🔑 TravelAdvantage Session

Paste your TA session cookie to authorize the server. Valid for ~3 days.

How to get the cookie:
1. Open traveladvantage.com/admin and log in
2. Press F12 → Application → Cookies → traveladvantage.com
3. Find the traveladvantage cookie
4. Copy the Value and paste it below

🔑 Freshdesk Session

Paste your full Freshdesk request cookie + CSRF token. Required for duplicate search, agent name resolution, and posting notes with inline images.

How to get them (one DevTools session captures both):
1. Open mwrlife.freshdesk.com and log in
2. Press F12 → Network tab → click any ticket so requests fire
3. Click any request to mwrlife.freshdesk.com → Headers → Request Headers
4. Copy the entire cookie: header value into Cookie below
5. Copy the x-csrf-token: header value into CSRF Token below (leave blank for read-only access)
